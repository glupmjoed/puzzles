Day 19: A Series of Tubes
=========================

https://adventofcode.com/2017/day/19

Part 1
------

The first part of the puzzle asks you in what order the characters `[A-Z]` appear
if you follow the path (starting from the top) in a diagram similar to the one
below. The path is made up of the characters `[-+|A-Z]`:

.example.txt
[source]
----
     |          
     |  +--+    
     A  |  C    
 F---|----E|--+ 
     |  |  |  D 
     +B-+  +--+ 
                
----

My solution consists of a link:walk.go[Go program] that walks the path and prints
it out in a linear fashion (+echo+ adds a newline to the output):

[source,shell]
$ ./bin/walk < example.txt && echo

.Output:
----
||A||+B-+|-|+--+C||+--+D+--|E----|---F
----

I then use the +tr+ command to delete any characters not in range +[A-Z]+:

[source,shell]
$ ./bin/walk < example.txt | tr -dc 'A-Z' && echo

.Output:
----
ABCDEF
----

Part 2
------

The second part of the puzzle asks you to find the length of the path. The
solution is obtained trivially using link:walk.go[] from part 1 and the `wc`
command to count the number of bytes:

[source,shell]
$ ./bin/walk < example.txt | wc -c

Output:
----
38
----
